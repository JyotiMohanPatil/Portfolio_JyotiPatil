Jyoti Patil – Multi-Page Portfolio (HTML/CSS/JS)
================================================

How to use
1) Replace placeholders (email, GitHub, LinkedIn) across pages.
2) Add your actual resume as /resume.pdf at the project root.
3) Replace project images with your own in /images and update pages.
4) Deploy to GitHub Pages or Netlify:
   - GitHub Pages: push the whole folder as a repo, then enable Pages.
   - Netlify: drag-and-drop the folder into the dashboard.

Pages
- index.html (About / Hero)
- projects.html
- skills.html
- education.html
- certificates.html
- experience.html
- contact.html

Theme
- Colors: navy background (#0F172A), teal accent (#14B8A6), white text (#F8FAFC)
- Cards: rounded corners, soft shadows, subtle gradients

Good luck! ✨
